Improvement case study
Six Sigma DMAIC — PT. Prima Jaya Fenolik Melamin, October 2024
Root cause 1
Operators changing machine parameters independently — no standardized settings per product type
Root cause 2
Operators not following SOPs consistently — process execution varied across shifts
Root cause 3
Incoming raw material quality inconsistent across batches from the same supplier
DefineProblem scoping
High defect rate (15–20%) and below-target output (~25,000 pcs/day) across 2 shifts
·Defined project scope: defect reduction and output improvement across 70 machines, 2 shifts
·Set targets: defect rate below 15%, output consistently at or above 30,000 pcs/day
MeasureData collection
Established baseline metrics before any changes were made
·Collected daily output, defect type, and downtime data per machine and per shift
·Tracked machine parameter settings across operators to identify variation patterns
·Logged raw material batch quality on arrival to identify correlation with defect spikes
AnalyzeRoot cause analysis
Identified three primary drivers of defects and output loss
·Parameter deviation by operators correlated directly with defect rate spikes on specific machines
·SOP non-compliance varied significantly between shifts — inconsistent process execution
·Batch-to-batch material quality variation identified as a trigger for unplanned defect increases
ImproveInterventions
Four concrete changes implemented in October 2024
·Standardized machine parameters per product type — operators prohibited from adjusting settings independently
·Implemented incoming material quality check before raw material enters the production floor
·Rescheduled production priorities based on available material quality — batches with quality issues deprioritized
·Rebalanced operator composition across shifts to distribute experienced operators more evenly
ControlSustaining the improvement
Monitoring systems put in place to detect regression early
·Built a real-time production KPI dashboard (Power BI) — tracking output, defect rate, downtime, and OEE daily; adopted by management as primary monitoring tool
·Process documentation updated and aligned with ISO 9001:2015 standards to formalize the new parameter standards and material handling procedures
